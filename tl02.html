Taller L02: Página de Animales por Categoría

Objetivo: Utilizar etiquetas HTML semánticas y de contenido para estructurar una página que clasifique animales según su dieta, basándose en la estructura de index.html.

Instrucciones del Ejercicio

Crea un nuevo archivo llamado animales.html y sigue los pasos a continuación. Deberás crear una carpeta recursos/animales/ y descargar allí imágenes de los animales que elijas.

  1. Paso 1: Estructura Base
    Crea la estructura fundamental del documento HTML (<!DOCTYPE html>, <html>, <head>, <body>) y asigna un título a la página.
  2. Paso 2: Encabezado Principal
    Dentro del <body>, añade un <header> con un título principal <h1> que diga "Animales del Mundo".
  3. Paso 3: Contenido Principal y Categorías
    Define la sección <main>. Dentro, crea tres secciones (<section>) para las categorías de animales. Cada sección debe tener un subtítulo <h2>:
    • Carnívoros
    • Herbívoros
    • Omnívoros
  4. Paso 4: Añadir Imágenes de Animales
    Dentro de cada sección, añade al menos dos imágenes (<img>) de animales que correspondan a esa categoría. Asegúrate de:
    • Usar rutas relativas correctas (ej: ./recursos/animales/leon.png).
    • Incluir un texto alternativo (alt) descriptivo para cada imagen.
    • Asignar un ancho (width) de "128" para mantener la consistencia visual.
    • Separar las imágenes con saltos de línea (<br>).
  5. Paso 5: Pie de Página
    Finalmente, crea un <footer> y añade un párrafo (<p>) con información de copyright o la fuente de las imágenes.

Solución Propuesta

A continuación se muestra el código HTML completo como una posible solución al ejercicio propuesto (archivo animales.html).


<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Animales del Mundo</title>
</head>
<body>

    <header>
        <h1>Animales del Mundo</h1>
    </header>

    <main>
        <section>
            <h2>Carnívoros</h2>
            <img src="./recursos/animales/leon.png" alt="Icono de un león" width="128">
            <br>
            <img src="./recursos/animales/tigre.png" alt="Icono de un tigre" width="128">
        </section>

        <section>
            <h2>Herbívoros</h2>
            <img src="./recursos/animales/elefante.png" alt="Icono de un elefante" width="128">
            <br>
            <img src="./recursos/animales/jirafa.png" alt="Icono de una jirafa" width="128">
        </section>

        <section>
            <h2>Omnívoros</h2>
            <img src="./recursos/animales/oso.png" alt="Icono de un oso" width="128">
            <br>
            <img src="./recursos/animales/cerdo.png" alt="Icono de un cerdo" width="128">
        </section>
    </main>

    <footer>
        <p>&copy; 2024 Animales del Mundo. Imágenes de Flaticon.</p>
    </footer>

</body>
</html>
